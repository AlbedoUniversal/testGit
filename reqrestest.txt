ЗАДАНИЕ:

Необходимо реализовать поиск пользователей по сервису GitHub.
Суть: 
    Вверху страницы инпут с кнопкой. Юзер вводит логин (или что угодно) в инпут и нажимает кнопку "Поиск".
    Внизу располагаются результаты поиска в виде карточек, на которых показан аватар юзера, его логин, ссылка на его профиль и рейтинг (это все присылает бэк)

Краткий алгоритм:
    1. Отправляешь запрос к API (см. документацию)
    2. Получаешь данные
    3. Размещаешь данные на странице

Доп.кейс:
    Необходимо сделать заглушку с надписью "Ничего не найдено" (или что угодно, на твой вкус), чтобы она появлялась только в том случае, если
    результаты поиска будут нулевыми.

Удачи! :)

Документация по заданию:

GET https://api.github.com/search/users?q=searchString - получить список юзеров

searchString - данные из инпута
